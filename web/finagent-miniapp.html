Мини-приложение для работы с таблицей

Таблица, контрагенты и проекты в одном окне

Этот экран можно открыть из кнопки Telegram Mini App. Он не заменяет Google Sheets, а даёт быстрый доступ к основным разделам и действиям.

Источник данных
Google Sheets
Подключается через ID таблицы
Режим
Telegram WebApp
Работает внутри бота
Статус
Готов к публикации
Нужен публичный URL
Быстрые действия
Разделы
Транзакции
Доходы, расходы, статьи, даты, суммы.
Основной лист
Контрагенты
Компании, CIF/NIF, адреса и заметки.
Справочник
Проекты
Связка с задачами, этапами и контрагентами.
Работа
Нажми любую кнопку. Дальше можно либо открыть Google Sheets по ссылке, либо отправить действие обратно в Telegram-бот.
Чтобы всё заработало, этот HTML нужно загрузить в репозиторий и опубликовать через GitHub Pages.
